# چک‌لیست کارهای روزانه — نسخه اندروید-محور (PWA)

اپلیکیشن وب بسیار روان، زیبا و مدرن مخصوص اندروید (و کامپیوتر) با تمرکز روی تجربه کاربری native-like.

## ویژگی‌های برجسته این نسخه

### 🎨 طراحی گرافیکی جذاب (Android Material feel)
- پالت رنگ زیبا در حالت روشن و **حالت شب کامل** (Dark Mode)
- کارت‌های با سایه نرم، گوشه‌های گرد بزرگ، انیمیشن‌های میکرو (smooth transitions)
- فونت‌های بهینه برای فارسی (Vazirmatn + سیستم فونت‌های اندروید)
- FAB (دکمه شناور +) کلاسیک اندروید برای افزودن سریع کار
- مودال افزودن کار به شکل Bottom Sheet نرم (لغزش از پایین)

### 🌙 حالت شب (Dark Mode)
- دکمه تغییر تم در هدر
- خودکار با تنظیمات سیستم (prefers-color-scheme)
- ذخیره ترجیح کاربر
- تغییر رنگ نوار وضعیت اندروید به صورت دینامیک

### 🚀 بسیار روان و بدون هنگ
- جاوااسکریپت خالص و سبک (هیچ فریم‌ورکی)
- انیمیشن‌های CSS با cubic-bezier طبیعی
- رندر سریع لیست کارها
- ذخیره خودکار با debounce
- تشخیص خودکار روز جدید حتی وقتی اپ باز است

### 📱 مخصوص اندروید (PWA)
- نصب واقعی روی هوم‌اسکرین (standalone)
- آیکون‌های باکیفیت + maskable
- Shortcuts در منوی طولانی فشار روی آیکون (افزودن کار سریع)
- کار کاملاً آفلاین
- مناسب برای نصب از کروم/فایرفاکس اندروید

### عملکرد اصلی
- چک‌لیست با کارهای **تکراری روزانه**
- ریست هوشمند در روز جدید
- یادداشت‌های سریع (ذخیره خودکار)
- یادآوری‌های زمان‌دار + نوتیفیکیشن مرورگر
- پیشرفت کارها (نوار + درصد)
- خروجی بک‌آپ JSON

## نحوه نصب و استفاده (اندروید)

1. فایل‌ها را در یک پوشه قرار دهید یا روی یک هاست رایگان آپلود کنید (Netlify, Vercel, GitHub Pages).
2. در مرورگر کروم یا فایرفاکس اندروید لینک را باز کنید.
3. از منوی ⋮ گزینه **«افزودن به صفحه اصلی»** یا **Install** را بزنید.
4. بعد از نصب، اپ از هوم‌اسکرین اجرا می‌شود (بدون نوار آدرس مرورگر).

**نکته:** برای بهترین تجربه، اپ را نصب کنید. نوتیفیکیشن‌ها در حالت نصب‌شده بهتر کار می‌کنند.

## فایل‌ها
```
index.html          ← اپ کامل (همه چیز داخل)
manifest.json       ← تنظیمات PWA + shortcuts اندروید
sw.js               ← آفلاین کردن اپ
icons/              ← آیکون‌ها (192 و 512)
README.md
```

## نکات فنی برای توسعه‌دهنده
- کاملاً client-side (localStorage)
- هیچ وابستگی خارجی به جز سرویس ورکر و manifest
- برای فونت بهتر: دستگاه‌های اندرویدی معمولاً فونت‌های فارسی خوبی دارند. اگر می‌خواهید فونت سفارشی، می‌توانید Vazirmatn را به صورت محلی اضافه کنید.
- برای نوتیفیکیشن‌های push واقعی (حتی وقتی اپ بسته است) نیاز به سرور + Firebase است.

## ساخت فایل نصبی واقعی اندروید (APK)

دو راه عالی آماده شده:

1. **سریع‌ترین راه (توصیه اکید)**: از سایت PWABuilder.com استفاده کنید — کمتر از ۲ دقیقه APK آماده می‌شود.
2. **راه حرفه‌ای با قابلیت‌های native بیشتر**: از پوشه `native/` (Capacitor) استفاده کنید.

راهنمای کامل + دستورالعمل قدم‌به‌قدم در فایل **BUILD_ANDROID_APK.md** نوشته شده.

همچنین لیست بلند پیشنهادهای آینده پروژه در فایل **SUGGESTIONS.md** موجود است.
